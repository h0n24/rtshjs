Real Time Syntax Highlighting JavaScript (RTSHJS)

This is a syntax highlighting engine written in JavaScript that colors text in real time while it is being typed in the browser.

I looked all around and didn't find anything that suits my needs so I began to code this. I found some other implementations of real-time coloring systems like spell checkers and a lot of online text editors (not real time), but they are so much more complex than this. This one is simple and has few lines of JavaScript code.

This is a part of a bigger project. I'm currently working on a online programming IDE (ecco.sourceforge.net) and this syntax highlighting engine will be its code editor.

This example shows syntax highlighting for Java and HTML. Other languages can be easily implemented, its a matter of some regular expressions. Colors used here are based on colors used by Eclipse IDE.
