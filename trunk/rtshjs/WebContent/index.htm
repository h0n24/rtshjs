Real Time Syntax Highlighting Code Editor JavaScript

This is a code editor with syntax highlighting written in JavaScript that colors text in real time while it's being typed in the browser.

Example shows syntax highlighting for Java and HTML. Other languages can be easily implemented, its a matter of some regular expressions. Colors used for Java are based on colors used by Eclipse IDE.
